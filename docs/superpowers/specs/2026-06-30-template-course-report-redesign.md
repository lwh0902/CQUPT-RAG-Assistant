## 4. 格式规范

- 页面：A4 纵向，采用模板页面边距和正式课程报告版式。

- 三级标题：五号宋体（10.5 pt），加粗。
- 正文：五号宋体（10.5 pt），固定行距 20 磅，首行缩进 2 字符。
- 英文、数字和英文标点：Times New Roman。
- 图题、表题：五号宋体，居中，统一编号。
- 参考文献：五号宋体；英文信息使用 Times New Roman。
- 页码：封面不显示，正文连续居中显示。
